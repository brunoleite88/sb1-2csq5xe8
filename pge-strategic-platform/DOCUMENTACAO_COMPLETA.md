## 🚀 COMO EXECUTAR O SISTEMA

### Pré-requisitos
- Docker e Docker Compose instalados
- 4GB RAM disponíveis
- 10GB de espaço em disco

### Passo a Passo

```bash
# 1. Clonar ou acessar o diretório do projeto
cd /workspace/pge-strategic-platform

# 2. Copiar arquivo de ambiente
cp .env.example .env

# 3. Editar variáveis de ambiente (opcional)
nano .env
# Principais variáveis:
# - MYSQL_ROOT_PASSWORD
# - JWT_SECRET
# - SPRING_DATASOURCE_URL

# 4. Iniciar todos os serviços
docker-compose up -d --build

# 5. Verificar status dos containers
docker-compose ps

# 6. Acessar o sistema
# Frontend: http://localhost
# API: http://localhost:8080/api
# Swagger: http://localhost:8080/swagger-ui.html
```

### Troubleshooting Comum

**Problema**: Container não inicia  
**Solução**: Verificar logs com `docker-compose logs backend`

**Problema**: Erro de conexão com banco  
**Solução**: Aguardar 30s para MySQL estar pronto ou verificar `.env`

**Problema**: JWT inválido  
**Solução**: Redefinir `JWT_SECRET` no `.env` e reiniciar

**Problema**: Porta 8080 ocupada  
**Solução**: Alterar porta no `application.yml` ou parar serviço conflitante
